# Absurd Generator

Generador de escenarios absurdos pero coherentes para simulación.

## Qué hace

Crea escenarios combinando:

* un dominio (logística, petróleo, etc.)
* una condición absurda
* un objetivo de entrenamiento
* una justificación coherente

Además genera un prompt visual listo para usar en herramientas de generación de imágenes.

## Uso

Abrir `index.html` y presionar "Generar escenario".
